## Program Explanation
This program made for beginner to senior user. (Usually undergraduated-graduated students)

This program can be splitted as two part.

**part 1. terminal mode**

This mode run on terminal, and default setting.
If you set the setting, you can just use Graphic-Interactive mode.

It'll be fast and lighter.

**part 2. Graphic-Interactive mode**

This mode use pop-up graphical window for user's convenience.

+ *Components | Graphs, Text-oriented Calculator(labels, buttons etc.)*

By using matplotlib, numpy, tkinter (for now plan.)
